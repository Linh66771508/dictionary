# 07 - Loi da gap va cach xu ly

## Loi Unicode path
- Duong dan D:\study\Đề án gay loi build.
- Da copy du an sang D:\study\dictionary.

## Loi symlink plugin
- PathExistsException trong linux\flutter\ephemeral.
- Xoa .dart_tool, build, windows\flutter\ephemeral, linux\flutter\ephemeral, macos\Flutter\ephemeral.
- Chay flutter clean va build lai.

## Loi backend khong ket noi
- App mo len bao ERR_CONNECTION_REFUSED.
- Nguyen nhan backend chua chay.
- Chay uvicorn hoac backend online.

## Loi Render deploy
- Khong tim thay requirements.txt o root.
- Can set rootDir=backend hoac dung render.yaml.

## Loi uvicorn khong tim thay
- Chay python -m uvicorn thay vi uvicorn.
