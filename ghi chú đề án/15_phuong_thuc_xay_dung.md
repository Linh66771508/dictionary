# 15 - Phuong thuc xay dung he thong

## 1. Thu thap yeu cau
- Xac dinh doi tuong su dung.
- Xac dinh tac vu chinh: tra cuu, dong nghia, chu de.
- Xac dinh cong cu va nen tang.

## 2. Thiet ke he thong
- Xay dung mo hinh client-server.
- Lua chon Flutter cho client.
- Lua chon FastAPI cho backend.
- Lua chon SQLite cho DB.

## 3. Thiet ke du lieu
- Tao schema: words, word_senses, topics, synonyms, proverbs.
- Dinh nghia quan he tu vung.

## 4. Trien khai backend
- Xay dung API theo REST.
- Test local bang uvicorn.
- Deploy Render.

## 5. Trien khai frontend
- Xay dung UI Flutter theo giao dien tham khao.
- Ket noi API.
- Build desktop va Android.

## 6. Kiem thu
- Test cac chuc nang tra cuu.
- Test them/xoa tu trong admin.
- Test API online Render.

## 7. Phat hanh
- Windows exe release.
- Android AAB.
