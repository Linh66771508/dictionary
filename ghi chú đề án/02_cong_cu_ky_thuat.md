# 02 - Cong cu va ky thuat su dung

## Frontend
- Flutter Desktop cho Windows.
- Flutter Android cho Google Play.
- UI tham khao tu bo giao dien Vietnamese Dictionary App UI va Admin UI for Dictionary App.

## Backend
- Python FastAPI.
- API REST tra cuu va quan tri du lieu.

## Database
- SQLite file.
- Tu dong tao schema khi backend khoi dong.

## Deployment
- Render Free Web Service.
- Backend chay online, co the sleep khi khong co truy cap.

## Cong cu ho tro
- Git va GitHub de luu tru ma nguon.
- Render dashboard de deploy.
- PowerShell de chay lenh.
- Visual Studio Build Tools de build Windows.

## Ly do chon
- Flutter: mot codebase cho Windows va Android.
- FastAPI: nhe, de dev, phu hop REST.
- SQLite: mien phi, khong can server DB.
- Render Free: deploy nhanh, khong mat phi.
